# 声笔自然（Shengbi Ziran）编码规则总结

### 1. 概述
声笔自然是经典“自然码”双拼输入法的**顶功（Dinggong）**改良版。它在保留自然码双拼方案的基础上，引入了顶功技术（即用笔画键作为编码的一部分或结束符），实现了字词分离、自动上屏和高效选重，大幅降低了学习难度并提高了输入效率。

### 2. 基础编码要素
声笔自然的编码由声母（s）、韵母（y）和笔画（b）三部分组成。

*   **声母（s）**：使用 `a`、`e`、`u`、`i`、`o` 以外的 21 个字母键。
    *   **平翘不分**：不区分平翘舌音，`z` 代表 `z/zh`，`c` 代表 `c/ch`，`s` 代表 `s/sh`。
        *   **与普通自然码的区别**：
            *   **普通自然码**：`zh`=**`v`**, `ch`=**`i`**, `sh`=**`u`**
            *   **声笔自然**：`zh`=**`z`**, `c`=**`c`**, `sh`=**`s`** (即平翘舌音合并)
    *   **零声母**：固定用 `v` 表示（如“爱”= `v` + `ai`）。
*   **韵母（y）**：沿用标准自然码的双拼韵母布局。
*   **笔画（b）**：仅使用 5 个元音键表示，大幅简化了传统辅助码。
    *   `a` = **折** (乛/乙)
    *   `e` = **横** (一)
    *   `u` = **撇** (丿)
    *   `i` = **竖** (丨)
    *   `o` = **点** (丶/捺)

### 3. 编码规则
符号说明：`1`、`2`、`3`、`0` 分别代表词组中的第1、2、3、末字。

#### 3.1 单字编码
*   **标准格式**：`s` + `y` + `b` + `b`
    *   即：`声母` + `韵母` + `首笔` + `次笔`
    *   示例：就 `jqoe`（`j`声+`q`韵+`o`点+`e`横）、钱 `qmue`
*   **扩展编码**：`s` + `y` + `b` + `b` + `b` + `b`
    *   当重码较多时，可继续追加第3、4笔画进行筛选。

#### 3.2 词组编码
词组采用不同于单字的编码空间，避免与单字重码。

*   **二字词**：`s1` + `y1` + `s2` + `y2`
    *   即：`字1声` + `字1韵` + `字2声` + `字2韵`
    *   示例：关系 `grxi`、正确 `zgqt`
*   **三字词**：`s1` + `s2` + `s3` + `y3`
    *   即：`字1声` + `字2声` + `字3声` + `字3韵`
    *   示例：示范区 `sfqu`
*   **多字词**：`s1` + `s2` + `s3` + `s0`
    *   即：`字1声` + `字2声` + `字3声` + `末字声`
    *   示例：中华人民共和国 `zhgg`（注意 `zh` 也是声母键位的一种，此处 `z`、`h`、`g`、`g`）
*   **词组扩展**：上述词组均可在末尾追加 **首字的前两笔** (`b1` + `b1`) 来快速离散重码。

### 4. 特色功能与简码

#### 4.1 选重与顶屏
*   **字母选重**：当输入达到4码且有重码时，`a`、`e`、`u`、`i`、`o` 键自动变为选择键（类似传统输入法的1-5数字键），手无需离开主键区即可选词。
*   **顶屏机制**：
    *   **单字顶屏**：单字仅需3码（声+韵+笔）即可将前一个字顶上屏幕。
    *   **简码顶屏**：部分静态简码（如三简字）在输入后续编码时自动上屏，无需空格。

#### 4.2 简码体系
*   **静态简码**：提供固定的一简、二简、三简字。
    *   一简：`j` (就)
    *   二简：`cs` (从)、`qi` (起)
    *   三简：`jxe` (节)
*   **空位简码**：利用空闲码位安排的高频组合。
    *   **声笔字**：`声` + `首笔` （如：`ra`=刃，`fe`=扶）
    *   **声声词**：`声1` + `声2` （如：`cx`=重新）

#### 4.3 动态码长
输入法会根据用户的使用频率动态调整词序。常用的四码词可能在下一次输入时仅需前三码或前两码即可首选上屏，从而缩短平均码长。

#### 4.4 反查功能
*   **笔画反查**：直接输入 `aeuio` 笔画序列。如 `uuao` 查“反”。
*   **两分反查**：`i` + 部件拼音。用于查询生僻字或非GBK字符。

#### 2. 技术说明

*   **完全本地计算**：所有算法都在本地实现，不依赖网络或外部包。
*   **依赖库**：仅使用 Python 标准库 (`re`, `sys`, `pathlib`)。
*   **数据来源**：所有数据都来自本地文件。

#### 3. 示例输出

```text
汉字     拼音         笔画数字       笔画字母       基础编码       完整编码           
----------------------------------------------------------------------------------------------------
水      shui       25         ia         sv         svia           
不      bu         13         eu         bu         bueu           
百      bai        13         eu         bl         bleu           
```

**注说明**：
1.  **黄色字体键位 (A, E, I, O, U, V)**：
    *   **A, E, U, I, O**：既是元音键，也根据上下文充当**笔画键**（折、横、撇、竖、点）或**选重键**。
    *   **V**：固定作为**零声母**（例如“爱”输入 `vai`），同时包含韵母 `ui`/`ü`。
2.  **声母合并**：
    *   **Z** 键对应全拼的 `z` 和 `zh`。
    *   **C** 键对应全拼的 `c` 和 `ch`。
    *   **S** 键对应全拼的 `s` 和 `sh`。
3.  **韵母布局**：完全遵循自然码双拼方案。
